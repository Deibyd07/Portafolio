# 📸 Imágenes de Proyectos

## Instrucciones para agregar capturas de pantalla

### 1. Toma capturas de pantalla de tus proyectos
- **Tamaño recomendado**: 1200x750px (ratio 16:10)
- **Formato**: JPG o PNG
- **Calidad**: Alta resolución pero optimizada para web

### 2. Nombra tus archivos según estos nombres:

```
images/
  ├── eventhub-preview.jpg      (Proyecto 3 - EventHub) ✅
  ├── project1-preview.jpg      (Proyecto 1 - Pendiente)
  ├── project2-preview.jpg      (Proyecto 2 - Pendiente)
  └── project4-preview.jpg      (Proyecto 4 - Pendiente)
```

### 3. Captura recomendaciones:

#### Para EventHub (eventhub-preview.jpg):
- Captura la página principal de https://gestion-de-eventos-omega.vercel.app/
- Muestra el dashboard o vista principal
- Asegúrate que se vea profesional

#### Para los otros proyectos:
- Captura la vista más representativa
- Si es posible, muestra la interfaz en acción
- Evita capturas con muchos colores brillantes que distraigan

### 4. Optimización de imágenes

Usa herramientas online para comprimir:
- **TinyPNG**: https://tinypng.com/
- **Squoosh**: https://squoosh.app/
- Objetivo: Menos de 200KB por imagen

### 5. Fallback automático

Si no tienes una imagen, se mostrará automáticamente un ícono SVG elegante.
¡No hay problema si aún no tienes todas las capturas!

---

## 🎨 Tips para buenas capturas:

✔️ Usa modo oscuro si tu app lo tiene
✔️ Captura en resolución 1920x1080 y recorta a 1200x750
✔️ Asegúrate que no haya información sensible
✔️ Idealmente muestra la UI con datos de ejemplo atractivos
✔️ Captura en un navegador limpio (sin extensiones visibles)

---

## 🚀 Cómo capturar screenshots profesionales:

### Opción 1: Screenshot nativo
- Windows: `Win + Shift + S`
- Mac: `Cmd + Shift + 4`

### Opción 2: Extensiones de navegador
- **Awesome Screenshot**: Captura página completa
- **GoFullPage**: Screenshot de toda la página

### Opción 3: Herramientas online
- **Screely**: https://screely.com/ (añade mockups de navegador)
- **Screenshots**: https://screenshots.cloud/ (mockups profesionales)
