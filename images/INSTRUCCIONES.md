# 🎯 Guía Rápida: Cómo Agregar Imágenes de tus Proyectos

## Para EventHub (Ya está conectado ✅)

### Paso 1: Captura una screenshot

Visita tu proyecto: **https://gestion-de-eventos-omega.vercel.app/**

1. Abre Chrome o Edge en modo incógnito (para que no se vean extensiones)
2. Ve a la página principal o dashboard de EventHub
3. Presiona `F12` para abrir DevTools
4. Presiona `Ctrl + Shift + P` (Windows) o `Cmd + Shift + P` (Mac)
5. Escribe "Capture screenshot" y selecciona "Capture full size screenshot"
6. Se descargará automáticamente

### Paso 2: Optimiza la imagen

**Opción A: Online (Más fácil)**
1. Ve a: https://tinypng.com/
2. Arrastra la imagen que descargaste
3. Descarga la versión comprimida

**Opción B: Recortar y redimensionar**
1. Ve a: https://www.iloveimg.com/resize-image
2. Sube tu screenshot
3. Redimensiona a: **1200 x 750 píxeles**
4. Descarga

### Paso 3: Renombra y coloca

1. Renombra el archivo a: `eventhub-preview.jpg`
2. Mueve el archivo a la carpeta: `Portafolio/images/`
3. ¡Listo! Recarga tu portafolio y verás la imagen

## 📸 Tips para Screenshots Profesionales

### ✅ Buenas prácticas:
- Captura con datos de ejemplo (no datos reales/sensibles)
- Usa un navegador limpio sin extensiones visibles
- Captura la vista más representativa del proyecto
- Asegúrate que el diseño se vea bien
- Comprime la imagen (menos de 200 KB idealmente)

### ❌ Evita:
- Capturas borrosas o de baja calidad
- Datos personales o sensibles visibles
- Errores de consola o warnings visibles
- Imágenes muy pesadas (+1MB)

## ¿No tienes imágenes todavía?

**¡No hay problema!** 

El portafolio mostrará automáticamente un ícono SVG elegante si no encuentra la imagen. Puedes agregar las imágenes más tarde sin romper nada.

---

## 🎨 Tamaño ideal de imágenes:

```
Resolución: 1200 x 750 píxeles (ratio 16:10)
Formato: JPG (mejor compresión) o PNG
Peso: < 200 KB (optimizado)
Calidad: 80-85% es suficiente
```

---

**¿Dudas?** Las imágenes se cargan automáticamente. Si cambiaste el nombre del archivo, asegúrate que coincida exactamente (incluyendo mayúsculas/minúsculas).
